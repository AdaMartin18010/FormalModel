# 形式化模型项目全局索引 / Formal Model Project Global Index

## 项目概述 / Project Overview

本项目是一个全面的形式化模型知识体系，涵盖基础理论、科学模型和行业应用，采用多语言实现（Rust、Haskell、Python、Julia等），为不同领域的模型提供标准化的数学描述和实现示例。
